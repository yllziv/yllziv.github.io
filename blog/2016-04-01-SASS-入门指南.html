SASS 入门指南

发表于 2016-04-01   |   分类于 web   |   阅读次数

安装

SASS是Ruby写的，先安装Ruby(添加PATH)，再安装Sass：gem install sass。
Compass是一套开源的基于Sass的CSS开发框架（了解即可），Sass是咖啡，则Compass是咖啡伴侣。
原则：写代码不要过于具体，达到要求就可以了；不要嵌套太多层代码；非必须情况不要写重复的混合宏。

Sass使用

后缀为.scss 意思是Sassy CSS。
在屏幕上显示.scss文件转换的css代码：
sass test.scss
如果要将显示结果保存成文件，后面再跟一个.css文件名：
　sass test.scss test.css
SASS提供四个编译风格的选项：

  • nested：嵌套缩进的css代码，它是默认值。
  • expanded：没有缩进的、扩展的css代码。
  • compact：简洁格式的css代码。
  • compressed：压缩后的css代码。（生产环境当中，一般选择这个）

sass –style compressed test.sass test.css
让SASS监听某个文件或目录，一旦源文件有变动，就自动生成编译后的版本。
　// watch a file
　　sass –watch input.scss:output.css
　　// watch a directory
　　sass –watch app/sass:public/stylesheets

注释

两种注释风格：

  1. 标准的CSS注释 / comment / ，会保留到编译后的文件
  2. 单行注释 // comment，只保留在SASS源文件中，编译后被省略。
  3. 在/后面加一个感叹号，表示这是”重要注释”。即使是压缩模式编译，也会保留这行注释，通常可以用于声明版权信息。/

调色函数

不能同时修改HSL和RGB的值，每种颜色属性调节方式都有相应的参数：
$red,$green,$blue:红/绿/蓝值在0-255之间
$hue:色调值在0-356之间的正数或者负数
$saturation,$lightness:饱和度和亮度是0-100%之间的正数或者负数
$alpha:透明度在0-1之间
